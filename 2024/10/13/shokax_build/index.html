把报错信息喂给 kimi：

Error: read ECONNRESET：

Error: read ECONNRESET 表示发生了一个错误，错误消息是 “read ECONNRESET”，意味着在尝试读取数据时，连接被对方重置。

{ errno: -4077, code: ‘ECONNRESET’, syscall: ‘read’ }：

这是一个 JavaScript 对象，提供了错误的额外信息。
errno: -4077 是一个系统特定的错误码，它通常与 ECONNRESET 错误相对应。
code: ‘ECONNRESET’ 是一个字符串，表示错误的类型。ECONNRESET 是一个标准的 Node.js 错误码，表明 TCP 连接被对方重置。
syscall: ‘read’ 表示在执行哪个系统调用时发生了错误。在这个情况下，是在尝试执行读取（read）操作时出现了问题。
